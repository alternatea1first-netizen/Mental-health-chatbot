# Mental-health-chatbot
This is a chatbot made to adress common mental health issues. This chatbot could answer questions which are related to mental health only. Please feel free to ask ur questions. This chatbot could be used as a reassurance and a medium which can give information regarding the issue of the user. 
This chatbot was made using IBM watson
The use of intents, dialogs and entities made it simpler to make this chatbot.
